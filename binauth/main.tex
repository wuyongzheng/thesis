\section{BinAuth: Secure Binary Authentication}
\label{sec:binauth}

\mypreamble{
This section presents BinAuth, which provides mandatory and efficient
authentication of binaries in Windows.
This joint work has been published in \cite{halim2008lightweight,wu2009esi}.
The author of this thesis focuses mainly on designing and implementing
the kernel mechanism and the user space signing tools.
The Software ID scheme and the hash-based message authentication code
are designed by other team members.
}
